Production Notes

started sculpting during coding bootcamp
during bootcamp covid began
working in non-drying oil clay
finished initial set during first dev job
started making sculptures more accurate
decided to choose scale to save decision-making
1:6 scale closest to size already working in
barbie / gi joe scale as bonus
took much longer than expected
completed initial 3 computers began molding process
not trained sculptor moldmaker etc
learned from model-making communities youtube etc
wife also artist taught silicone mold-making
eventually found right silicone for molds
made molds without vacuum chamber
turned out much better than expected
used OOMOO 25 from Smooth-On (company)
no bubbles / imperfections
cast master set in resin
used Smooth Cast 300 also from Smooth-On
flawless no-bubble casts no pressure pot
no training or experience with concrete
had difficult time finding resources
used Quick-Crete for casting Home Depot
sand aggregate of varying grain size
different balances of flow-release and retarder
used dry pigments to darken and color
meticulous proportion notes with each cast
about 20-30 casts made in first run
various colors, material proportions etc
initial molds began deteriorating new set needed
experimented for right sealant for concrete
shellac for initial coat overly glossy
matte water-based urethane as temp fix
> put foam pads on bottoms for desk use
found appropriate shipping boxes and packing materials
made readme and soc-media pages
began mailing to devs artists friends
